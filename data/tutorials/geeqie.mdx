## What is Geeqie?

The Geeqie program is an open-source freeware image viewer and image organizer for Unix-based systems, including Linux and Apple's OS X. It was initially developed as a fork of GQview, which had ceased development. The application uses the GTK toolkit. Geeqie has many features focused on speed and organizing. A high-speed preview for many formats is supported. Geeqie allows Easy tagging by editing and viewing comments and EXIF data.

## Geeqie JXL support

Geeqie fully supports JPEG XL. You can easily view and edit any JPEG XL image as you would do with any other image format. You can <Link href="www.geeqie.org/" text="download Geeqie on its official homepage"/>.
